# 🎯 Number Guessing Game

A simple and interactive web-based game where players try to guess a randomly generated number within a limited number of attempts. This project was built using **HTML, CSS, and JavaScript** and demonstrates basic concepts of web development and game logic.

---

## 📖 About the Project

The **Number Guessing Game** is designed to be a fun and beginner-friendly web application. The computer randomly generates a number within a specific range, and the player must guess the correct number.

After each guess, the game provides feedback such as **"Too High"** or **"Too Low"**, helping the player get closer to the correct answer. The player must guess the number before running out of attempts.

This project is useful for understanding:

* JavaScript logic and functions
* DOM manipulation
* Event handling
* Local storage for saving scores
* Basic UI design using CSS

---

## 🚀 Features

* 🎮 Random number generation
* 🎯 Difficulty levels (Easy, Medium, Hard)
* 🔢 Limited number of attempts
* 📜 Guess history tracking
* 🔁 Restart game option
* 🏆 Best score tracking using browser local storage
* 🎨 Clean and responsive user interface

## ▶️ How to Run the Project

You can run this project easily in your browser.

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html**.
4. The game will open in your browser.

You can also run it using **VS Code Live Server** for development.

---

## 🎮 How to Play

1. Select a difficulty level.
2. Enter your guess in the input box.
3. Click **Submit Guess**.
4. The game will tell you whether your guess is **too high or too low**.
5. Continue guessing until you find the correct number or run out of attempts.
